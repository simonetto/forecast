# Renzo's Forecast

A simple weather forecast page using Angular 1.5 and ES6

## Installation

```
$ git clone git@github.com:simonetto/forecast.git
$ npm install
```

## Development server

```
$ npm run serve
```

## Testing

```
$ npm run test
```

## Building the application

```
$ npm run build
```
